# Dagster Open Source vs Cloud

Dagster Open Source and Dagster Cloud - what's the difference?

In this guide, we'll walk you through each of these offerings, including a side-by-side comparison.

---

## Dagster Open Source

Available (and free!) for anyone to use, Dagster itself is an open source data orchestration platform built for data teams.

With Dagster Open Source, you manage the operational needs to run and deploy Dagster, including infrastructure and system upgrades.

---

## Dagster Cloud

Dagster Cloud builds upon the Dagster you know and (hopefully!) love.

A fully-managed service, Dagster Cloud allows you to offload operational needs and focus on development. With the same user-friendly UI as Dagster Open Source, Dagster Cloud is both easy-to-use and scalable.

Along with all Dagster Open Source functionality, Dagster Cloud also offers:

- Two deployment options - [Serverless](/dagster-cloud/deployment/serverless) and [Hybrid](/dagster-cloud/deployment/hybrid), allowing you to choose between a fully-hosted solution and one where you bring-your-own-compute via an [agent](/dagster-cloud/deployment/agents)
- Automatic system upgrades
- Integrated CI/CD capabilities via [Branch deployments](/dagster-cloud/developing-testing/branch-deployments)
- [Role-based access control](/dagster-cloud/account/managing-users)
- [Single Sign-on (SSO)](/dagster-cloud/account/authentication)
- Additional support options

Check out the [side-by-side comparison](#side-by-side-comparison) for the full list of features, or the [Dagster Cloud docs](/dagster-cloud) to get started.
